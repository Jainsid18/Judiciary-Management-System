# Judiciary Management System

## Overview
A role-based court case management system built using Spring Boot.

## Features
- JWT Authentication
- Role Based Authorization
- Case Filing
- Judge Assignment
- Hearing Scheduling
- Verdict Management
- Document Upload
- Notifications
- Audit Trail
- Dashboard Analytics

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Swagger/OpenAPI

## Roles
- Admin
- Judge
- Citizen
